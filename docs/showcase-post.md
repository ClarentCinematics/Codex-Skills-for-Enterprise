# Showcase Post

Codex Skills for Enterprise `v0.1.0` is live: a validated, MIT-licensed catalog of enterprise-ready Codex skills for executive operations, engineering operations, revenue operations, and knowledge operations.

The repository is built to look and operate like a real skill system, not a loose prompt dump:

- 15 curated skills across 4 operating packs.
- Registry-backed catalog metadata in `skill-registry.json`.
- Generated public catalog in `docs/catalog.md`.
- CI validation for skill structure, registry consistency, generated docs, smoke fixtures, and Python syntax.
- Sample outputs that show expected shape, caveats, and "must not invent" behavior.
- Level 3 script-assisted helpers for CI failure triage, CRM hygiene auditing, and knowledge-base metadata checks.

Featured workflows:

- `meeting-intelligence`: turn meeting notes into decisions, owners, risks, and follow-ups.
- `ci-failure-triage`: extract the primary failure signal from noisy CI logs.
- `crm-hygiene-auditor`: find ownership, activity, duplicate, and forecast hygiene risks.
- `knowledge-base-capture`: convert working knowledge into reviewable KB articles.
- `cao-operating-pulse`: produce board-ready Chief AI Officer operating pulse briefs.
