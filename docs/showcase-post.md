# Showcase Post

Codex Skills for Enterprise is a validated, MIT-licensed catalog of enterprise-ready Codex skills for executive operations, engineering operations, revenue operations, knowledge operations, reliability, customer operations, data quality, product delivery, and risk governance.

The repository is built to look and operate like a real skill system, not a loose prompt dump:

- 20 curated skills across 9 operating packs.
- Registry-backed catalog metadata in `skill-registry.json`.
- Generated public catalog in `docs/catalog.md`.
- CI validation for skill structure, registry consistency, generated docs, smoke fixtures, and Python syntax.
- Sample outputs that show expected shape, caveats, and "must not invent" behavior.
- Level 3 script-assisted helpers for CI failure triage, CRM hygiene auditing, knowledge-base metadata checks, incident postmortems, support deflection, data-quality triage, requirements readiness, and vendor security review.

Featured workflows:

- `meeting-intelligence`: turn meeting notes into decisions, owners, risks, and follow-ups.
- `ci-failure-triage`: extract the primary failure signal from noisy CI logs.
- `crm-hygiene-auditor`: find ownership, activity, duplicate, and forecast hygiene risks.
- `knowledge-base-capture`: convert working knowledge into reviewable KB articles.
- `cao-operating-pulse`: produce board-ready Chief AI Officer operating pulse briefs.
- `incident-postmortem-assistant`: turn timelines into blameless incident reviews.
- `support-deflection-miner`: mine repeated support themes into KB and automation candidates.
- `data-quality-triage`: check CSV quality before dashboards or automation rely on it.
- `requirements-to-acceptance-criteria`: convert rough specs into testable delivery criteria.
- `vendor-security-review`: find vendor security evidence gaps before approval review.
