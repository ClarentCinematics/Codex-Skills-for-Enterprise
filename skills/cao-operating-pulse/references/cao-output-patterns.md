# CAO Output Patterns

## Board Quarterly Brief

Compress the full pulse for a board or exec committee read.

```text
AI Weather Report
30-Second Board Slide
Wins That Survived Contact With Reality
Portfolio: Scale / Fund / Watch / Park / Retire
Top 3 Risks (with mitigations)
Investment Ask or Reallocation
Decisions Needed Before Next Meeting
```

## Vendor Circus Review

Use when comparing vendors, copilots, model providers, or platform pitches.

| Signal | Question to answer |
| --- | --- |
| **Claim** | What are they promising? |
| **Evidence** | What proof exists in your environment? |
| **Lock-in** | Data, workflow, contract, or model dependency? |
| **Control fit** | Security, privacy, audit, residency, human review? |
| **Pilot cost** | Time, money, and political capital to learn something real? |
| **Verdict** | `Proceed` / `Pilot narrow` / `Watch` / `Pass` |

End with a **Vendor Circus Score** from 1 (substance) to 5 (pure performance art) and the smallest experiment that would falsify the pitch.

## Incident Post-Mortem Pulse

Use after model failures, prompt leaks, bad outputs, policy breaches, or reputational AI events.

```text
What Happened (facts only)
Customer / Business Impact
Root Cause Hypotheses (labeled)
Control Failures
Communications Needed
Fix Now / Fix Next / Prevent Forever
Board or Regulator Sensitivity
```

Keep tone crisp. No blame theatre. Name owners.

## AI Literacy Rollout

Use when planning enablement, champions, training, or adoption programs.

```text
Audience Segments
Jobs-To-Be-Done With AI
Current Friction
Enablement Bets (small, medium, bold)
Guardrails To Teach Early
Success Signals
What Not To Teach Yet
```

## Shadow AI Bingo

Use when informal AI usage is part of the input. Mark only squares supported by evidence.

```text
[ ] Customer data in a public chatbot
[ ] Code pasted into unsanctioned tools
[ ] "Just this once" workflow bypass
[ ] Spreadsheet sent to an unknown SaaS
[ ] Executive demo became production
[ ] No owner for the workaround
[ ] Legal/procurement never consulted
[ ] Team thinks IT already approved it
```

If three or more squares are checked, escalate governance and enablement together — bans alone rarely work.
